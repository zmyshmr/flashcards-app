# 単語帳Webアプリ (Flashcards App)

いつでもどこでも、スマホ（iPhone）で手軽にサクッと学習できる、モバイルファーストな単語帳Webアプリです。

👉 **[アプリを開く (GitHub Pages)](https://zmyshmr.github.io/flashcards-app/)**

## 🌟 主な特徴

- **スマホに最適化されたUI**: ネイティブアプリのような洗練されたダークテーマと、心地よいカードめくり（Flip）アニメーション。
- **一括登録に対応**: 単語の個別登録はもちろん、Excel等から「問題,答え」の形式でコピペして一気に複数登録することが可能です。
- **学習履歴のトラッキング**: 「⭕️ 覚えてた」「❌ 忘れてた」を自己判定することで、学習履歴が蓄積。全体の正答率や、各単語の直近の成績（最大5回分）が一覧画面でひと目で確認できます。
- **セキュア＆高速**: バックエンドサーバーやデータベースを持たないSPA（Single Page Application）です。データはすべてブラウザの `localStorage` に保存されるため、オフラインに近い感覚で高速に動作し、個人データが外部に漏れる心配がありません。

## 📱 オススメの設定方法 (iPhone)

iPhoneのSafariでアプリを開き、画面下部の「共有ボタン」から**「ホーム画面に追加」**を選択してください。URLバーなどが非表示になり、フルスクリーンのネイティブアプリのような感覚でご利用いただけます。
