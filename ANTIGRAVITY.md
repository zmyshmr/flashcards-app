# プロジェクト憲法 (ANTIGRAVITY.md)

## 1. プロジェクト概要
- **プロジェクト名**: 単語帳Webアプリ (仮)
- **プラットフォーム**: Webブラウザ (特にiPhoneでの利用を想定)
- **ホスティング**: GitHub Pages (静的サイト)

## 2. 要件定義
- **目的**: 英語、プログラミング用語、業務マニュアルなど、あらゆる用途に使える汎用的な単語帳。個別登録と一括登録（将来実装予定含む）に対応。
- **ターゲットユーザー**: 自分専用
- **利用シーン**: いつでもどこでも、スキマ時間（特にiPhoneなどのスマホ）で利用。

## 3. MVP（最小構成）の範囲
第一段階（味見フェーズ）として、まずは以下の「動く体験」を作ることを目指します。
1. **単語の個別登録**: 表面（問題）と裏面（答え）を入力して追加できる機能。
2. **学習モード**: 単語カードを1枚ずつ表示し、タップで裏返る（めくる）機能。
3. **データ保存（ローカル）**: サーバーやDBを使わず、ブラウザの `localStorage` を使用してデータを保存（GitHub Pagesでの静的ホスティングと相性が良いため）。

※「一括登録」機能は、MVPの動作確認が完了した直後の次のステップ（Phase 2）で実装します。

## 4. 技術スタック
- フロントエンド: HTML, Vanilla CSS, JavaScript
- データ保存: localStorage (ブラウザローカル保存)
- サーバー: GitHub Pages (静的サイト)
